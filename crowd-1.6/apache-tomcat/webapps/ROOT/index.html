Crowd — Identity Management Simplified

Crowd is an application security framework that handles authentication and authorization calls for your web-applications. With Crowd you can quickly integrate web-applications into a single security architecture that supports single sign-on and centralized identity management.

The application is divided into two parts:

  • The administration console is a clean and powerful web-interface to manage directories, users and their security rights.
  • The integration API provides a platform neutral way to integrate web-applications into a single security architecture. With the integration API applications can quickly access user information or perform security checks.

The first time you access the Crowd administration console, you will be walked through the initial configuration of your deployment.

OpenID Server

The Crowd OpenID Server implements the OpenID specification. When using your Crowd OpenID login, you can have the same username and password for the public websites you use that support OpenID.

Demo Application

The demo application highlights best practices when using the Crowd framework. The Crowd download archive contains the entire source to the demo application, which can be used as an example when integrating your web-applications.

OpenID Client Example

The OpenID demo application can be used with the Crowd OpenID server as an example of how to use OpenID. The Crowd download archive contains the entire source to the OpenID demo application, which can be used as an example when integrating your web-applications.
